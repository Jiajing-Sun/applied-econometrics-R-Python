# Chapter 03: 基本概率论与统计推断

本章不依赖外部数据源。教材变量使用掷骰子、Monte Carlo 模拟和均值检验来说明估计量、抽样分布、中心极限定理、假设检验与置信区间；中文配套代码保留这些教学对象，并把临时输出整理成可复现文件。

## 本章保留的教材方法

- 掷骰子样本均值模拟
- 10,000 次 Monte Carlo 重复实验
- 样本均值抽样分布的直方图
- 中心极限定理：均匀分布与卡方分布的对比
- 均衡骰子假设下的一样本 t 检验
- 95% 置信区间

## 运行方式

```bash
Rscript R/chapter03.R
python3 python/chapter03.py
```

## 输出文件

- `figures/`：Monte Carlo 直方图、中心极限定理图、频数与置信区间图，全部使用中文标签。
- `tables/`：50 次掷骰结果频数表，包含 CSV 与可粘回 LaTeX 的 `.tex` 表。
- `results/`：模拟分布、Monte Carlo 摘要、t 检验结果和结果说明。
